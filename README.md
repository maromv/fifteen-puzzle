# 15 Puzzle game
For now - terminal based app
You can enter tile value that you want to swap the blank one with,
or use arrow keys "controlling" void tile

## Setup:
project is pnpm based and keeps lock files related, yet you can use plain npm

### pnpm
```bash
pnpm install
pnpm start
```

### npm
```bash
npm i
npm start
```
### to get even more fun
turn on emoji using
```bash
npm start --emoji
```

to fill your void with a friendly ghost
